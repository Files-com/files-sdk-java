# Files.Models.Status

## Example Status Object

```
{
  "code": 200,
  "message": "",
  "status": "",
  "data": "",
  "errors": [
    {
      "fields": [

      ],
      "messages": [

      ]
    }
  ]
}
```

* `code` / `code`  (int64): Status http code
* `message` / `message`  (string): Error message
* `status` / `status`  (string): Status message
* `data` / `data` : Additional data
* `errors` / `errors`  (array): A list of api errors
