# Files.Models.ChatMessage

## Example ChatMessage Object

```
{
  "id": 1,
  "role": "example",
  "content": "example",
  "created_at": "2000-01-01T01:00:00Z"
}
```

* `id` / `id`  (int64): Chat Message ID.
* `role` / `role`  (string): Message role.
* `content` / `content`  (string): Message content.
* `created_at` / `createdAt`  (date-time): Message creation date/time.
