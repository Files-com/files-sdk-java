# Files.Models.AgentPushUpdate

## Example AgentPushUpdate Object

```
{
  "version": "example",
  "message": "example",
  "current_version": "example",
  "pending_version": "example",
  "last_error": "example",
  "error": "example"
}
```

* `version` / `version`  (string): Pushed agent version
* `message` / `message`  (string): Update accepted or reason
* `current_version` / `currentVersion`  (string): Installed agent version
* `pending_version` / `pendingVersion`  (string): Pending agent version or null
* `last_error` / `lastError`  (string): Last error message or null
* `error` / `error`  (string): Error code
